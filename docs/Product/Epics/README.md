# Epics

Version : 1.0

Projet : ADCI

---

# Objectif

Les Epics représentent les grands domaines fonctionnels du produit ADCI.

Ils servent à organiser le Product Backlog et à planifier les différentes versions de la plateforme.

---

# EPIC-01 — Gestion des adhésions

## Description

Permettre à un visiteur de réaliser une demande d'adhésion au mouvement ADCI.

## Valeur métier

Digitaliser le processus d'adhésion.

## Version

V1

## User Stories

- US-01 Consulter la page Adhésion
- US-02 Renseigner les informations personnelles
- US-03 Ajouter une pièce d'identité
- US-04 Ajouter un conjoint
- US-05 Choisir une formule
- US-06 Vérifier le récapitulatif
- US-07 Soumettre la demande
- US-08 Utiliser le site sur mobile
